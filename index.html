SongSearch

An Aggregate Web Search Tool for DJs to use on the fly.

Built with AngularJS, Browserify, Firebase, Bootstrap, and Materialize, this app allows users to search multiple music databases at once. The app grabs song data from iTunes using the iTunes API, and other music sites with custom-built web scrapers.

MowFlow

A Web App for landscape professionals to plan and optimize their schedules.

Built with Ruby on Rails, this app allows users to harness the power of data analysis and generate the most efficient work routing schedules. The location of the clients alongside other job attributes are optimized by generating and analyzing the data with a custom-tailored python algorithm using the Google OR-Tools library and various APIs to solve the “Traveling Salesman Problem.”
