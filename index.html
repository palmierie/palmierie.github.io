Projects

SongSearch

An app that I built for DJs. Allows users to search multiple music databases at once. Built with AngularJS, Browserify, Firebase, Bootstrap, and Materialize. App grabs song data from iTunes using the iTunes API, Beatport using a custom-built javascript webscrapper, and Headliners Music Club using a custom-built JQuery webscrapper. With current version, users must use a CORS Origin blocker, such as Moesif.

SongSearch Github Repo

MowFlow

An app that I built for small landscaping companies. Allows users to enter their client list and generate the most efficient mow schedules. Built with Ruby on Rails, Python, Google OR-Tools, OSRM Api, Google Maps Api, and Bootstrap,. App grabs map data from Google Maps using the Google Maps API. The app features custom-built algorithm to solve the Traveling Salesman Problem with parameters that make the most sense to mow scheduling.
