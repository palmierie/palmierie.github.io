Projects

SongSearch

Built with AngularJS, Browserify, Firebase, Bootstrap, and Materialize, this app allows users to search multiple music databases at once. The app grabs song data from iTunes using the iTunes API, Beatport (online music store) using a custom-built javascript web scraper, and Headliners Music Club (online record pool subscription service) using a custom-built JQuery web scraper. Results are displayed in a list with sort options, buy links, and save options (for signed in users). Signed in users can view a list of saved songs and toggle which music databases to search.

MowFlow

Built with Ruby on Rails, this app allows users to enter job information for each client and generate the most efficient work routing schedules. Coordinates from Google Places API for each client are sorted by job date and sent to the OSRM API to return a distance matrix. The distance matrix alongside job durations entered by the user are formatted and then ran through a custom-tailored python algorithm using the Google OR-Tools library to solve the “Traveling Salesman Problem” and the most efficient driving routes are returned. Users can reschedule jobs using the drag and drop calendar week view, view all jobs on a calendar month view or list view, view a client list, print a schedule and get Google Maps route information.
